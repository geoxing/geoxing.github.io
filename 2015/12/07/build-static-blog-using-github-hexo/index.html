##command summary

  1. general command

    hexo new postName
    hexo new page pageName
    hexo generate
    hexo server
    hexo deploy
    hexo help
    hexo version

  2. composed command

    hexo deploy
    hexo server -g

  3. simplify:

    hexo n == hexo new
    hexo g == hexo generate
    hexo s == hexo server
    hexo d == hexo deploy

2.6 sitemap
2.6.1 you can install sitemap plugin,commit your site to google,

$ npm install hexo-generator-sitemap

2.6.2 add in the Hexo_config.yml:

# Extensions
Plugins:
hexo-generator-sitemap
#sitemap
sitemap:
path: sitemap.xml

2.6.3 visit http://localhost:4000/sitemap.xml, you can get sitemap,file path \sitemap.xml
2.6.4 show the sitemap
