#### Sitemap    
1. you can install  sitemap plugin,commit your site to google
``` 
  $ npm install hexo-generator-sitemap  
``` 
-  add in the Hexo\_config.yml:
  >\# Extensions  
Plugins:  
   hexo-generator-sitemap    
  \#sitemap   
sitemap:  
  path: sitemap.xml  
  
2. visit http://localhost:4000/sitemap.xml, you can get sitemap,file path \sitemap.xml  
3. show the sitemap

  in the themes/pacman/ _config.yml,add 
  >menu:
  Home: /
  Archives: /archives
  Rss: /atom.xml
  Sitemap: /sitemap.xml
  
#### RSS  
1. install feed plugin,file path \atom.xml

$ npm install hexo-generator-feed
```

  1. seal off ，edit Hexo_config.yml,add

    # Extensions
    Plugins:

    • hexo-generator-feed
    • hexo-generator-sitemap

    #Feed Atom
    feed:

    • type: atom
    • path: atom.xml
    • limit: 20
